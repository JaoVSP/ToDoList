# Tech Stack

* React Native
* Expo
* Redux


# Installation

```
git clone https://github.com/JaoVSP/ToDoList.git
cd ToDoList
npm install
```

# Run

```
npm start
```

# Mobile Build

```
npx expo run:android
npx expo run:ios
```

# After the first build

```
npx expo start
```
